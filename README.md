# Clinique Numerisk – Landing Page & Démonstration Logiciel

Bienvenue dans le projet **Clinique Numerisk**, une landing page commerciale conçue pour présenter et promouvoir le logiciel de gestion des cliniques.  
Cette application est développée avec **Vue 3**, **Tailwind CSS**, **Vue Router** et **AOS (Animations au scroll)**.

---

## 1️⃣ Objectif du projet

Cette landing page sert à :

- Présenter le logiciel aux clients potentiels de manière claire et attractive.  
- Mettre en avant les problèmes rencontrés par les cliniques et montrer comment le logiciel y répond.  
- Convaincre les clients grâce à des textes persuasifs, des icônes et illustrations professionnelles.  
- Permettre aux visiteurs de demander une **démonstration gratuite** via un formulaire intégré.  

---

## 2️⃣ Technologie utilisées

- **Frontend :** Vue 3, Tailwind CSS, Vue Router  
- **Animations :** AOS (Animate On Scroll)  
- **Icônes :** Heroicons (SVG, faciles à intégrer dans Vue 3)  
- **Images :** Illustrations pour HeroSection et CTA pour rendre la page visuellement attractive  
- **Déploiement :** Serveur web classique (Apache/Nginx) ou Vite en local pour le développement

---

## 3️⃣ Couleurs principales de la marque

La landing page utilise les couleurs **brandées** de Numerisk :

- Orange : `#f48024` → `primaryOrange` dans Tailwind  
- Violet : `#5e35b1` → `primaryPurple` dans Tailwind
